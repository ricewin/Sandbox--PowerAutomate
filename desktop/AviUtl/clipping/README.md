# AviUtl - Clipping A

## AviUtl を操作し、拡張編集から動画ファイルを追加、クリッピングするフローです

### 動作確認

- Windows 11
- AviUtl ver. 1.10
- 拡張編集 ver. 0.92

### 使用方法

1. クリッピングで参照する動画を準備してください。
1. AviUtl を起動し、ベースになる動画をフォワードにしてください。
1. 実行します。

### 動画ファイルについて

参照する動画を変更する場合は、Main フローの「フォルダー内のファイルを取得」プロパティで

ファイルフィルター: `*.avi`

の部分を任意のフィルターに変更してください。
